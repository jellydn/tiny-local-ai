BASIC COMMANDS
══════════════════════════════════════════════════════════════════════

# Run complete benchmark (all 15 prompts, both models)
python3 scripts/benchmark.py

# Just preview what will run (no servers started)
python3 scripts/benchmark.py --dry-run

# Test only coding prompts
python3 scripts/benchmark.py --categories coding

# Test only GLM-4.7-Flash (skip Qwen3)
python3 scripts/benchmark.py --skip-model qwen

# Save results to a specific file
python3 scripts/benchmark.py --output my-results.json


WHAT IT DOES
══════════════════════════════════════════════════════════════════════

1. Starts Qwen3-Coder-Next server (port 8000)
2. Runs 15 test prompts against Qwen3
3. Stops Qwen3 server
4. Starts GLM-4.7-Flash server (port 8001)
5. Runs same 15 prompts against GLM-4.7-Flash
6. Stops GLM server
7. Generates comparison report
8. Saves results to JSON


UNDERSTANDING THE RESULTS
══════════════════════════════════════════════════════════════════════

Time(s):     How long the prompt took to respond
Tokens:      Number of tokens in the response
Tokens/s:    How many tokens per second (throughput)
Status:      "success" or "error"

Higher tokens/s = Faster model ✓
Lower response time = Better for real-time tasks ✓


EXPECTED TIMES
══════════════════════════════════════════════════════════════════════

Full benchmark (15 prompts × 2 models):
  • ~10-15 minutes total

Coding category only (5 prompts × 2 models):
  • ~3-5 minutes total

Single model (5 prompts):
  • ~1-2 minutes

Note: Initial model load is slower. Subsequent prompts are faster.


TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════

Problem:  "Model not found" error
Solution: Ensure models are in ~/Library/Caches/llama.cpp/

Problem:  Server doesn't start
Solution: Check that /opt/homebrew/bin/llama-server exists

Problem:  Out of memory
Solution: Use --skip-model to test one model at a time

Problem:  Very slow / hangs
Solution: Reduce timeout: --timeout 60

    data = json.load(f)
    for model in data['models']:
        print(f\"\n{model['model_name']}\")
        print(f\"  Avg time: {model['summary']['avg_time']:.2f}s\")

"

CUSTOMIZING TEST PROMPTS
══════════════════════════════════════════════════════════════════════

Edit scripts/benchmark-prompts.json to:
  • Add new test questions
  • Change existing prompts
  • Add custom categories

Format:
{
  "id": "custom_001",
  "category": "coding",        # coding, qa, or reasoning
  "difficulty": "medium",      # easy, medium, hard
  "prompt": "Your question",
  "expected_tokens": 100
}
